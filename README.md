# Mey Beauty Paris — React (Vite)

## Démarrer

1. Installer les dépendances

```bash
npm install
```

2. Lancer le serveur

```bash
npm run dev
```

## Notes

- Le CSS d'origine a été conservé dans `src/styles.css`.
- La page a été découpée en composants React dans `src/components/`.
- Les visuels (placeholders SVG) ont été remplacés par des images professionnelles (onglerie/cosmétique) via URLs.
